# 🍽️ Sanag Restaurant

A modern and responsive restaurant website built using **HTML5** and **CSS3**. The website highlights authentic Somali cuisine with an elegant, user-friendly design that works seamlessly across desktop, tablet, and mobile devices.

## 🌟 Features

* 🏠 Beautiful Home Page
* 🍴 Restaurant Menu
* 👨‍🍳 About Us Section
* 📞 Contact Information
* 📱 Fully Responsive Design
* 🎨 Clean and Modern User Interface
* ✨ Smooth Hover Effects & Animations
* ⚡ Fast Loading and Lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* Responsive Web Design
* Flexbox
* CSS Grid
